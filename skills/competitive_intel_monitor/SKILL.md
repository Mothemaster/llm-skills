# Skill: Competitive Intelligence Monitor

## Purpose
Track and report ongoing changes in competitors' pricing, positioning, campaigns, and messaging — not a one-time snapshot, but a continuous monitoring mindset.

## When to use this skill
- User asks "what are my competitors doing," "track competitor pricing," "monitor competitor campaigns," or "compare us to X."

## Process
1. Identify the competitor(s) named or inferred from context.
2. Research current: pricing/plans, homepage messaging, recent campaigns or promotions, product updates, customer reviews sentiment.
3. Compare each finding against the user's own brand/product using a structured table.
4. Flag any recent changes (last 30-90 days) as "New" or "Changed" — these matter most.
5. Highlight gaps: what competitors offer that the user doesn't, and vice versa (unique advantages).
6. Recommend 2-3 concrete actions based on findings (e.g., "adjust pricing tier," "counter this messaging angle," "address this feature gap").

## Output format
```
## Competitor: [Name]
| Dimension | Them | Us | Gap/Opportunity |
|---|---|---|---|
| Pricing | ... | ... | ... |
| Messaging | ... | ... | ... |
| Recent moves | ... | ... | ... |

## Recommended actions
1. ...
2. ...
```

## Quality checklist
- [ ] At least 2 competitors covered if multiple exist
- [ ] Recent changes (30-90 days) explicitly flagged
- [ ] Every finding tied to a specific, actionable recommendation
- [ ] No stale data presented as current without a date reference
