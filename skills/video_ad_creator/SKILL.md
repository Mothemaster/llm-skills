# Skill: Video Ad Creator

## Purpose
Write structured scripts and shot lists for short-form video ads (TikTok, Reels, YouTube Shorts) optimized for the first 3 seconds and conversion.

## When to use this skill
- User asks for "a video ad," "a TikTok script," "a Reels ad," "UGC-style video script."

## Process
1. Define the hook (first 3 seconds) — must stop the scroll. Use a bold claim, question, or visual surprise.
2. Define the problem/pain point the audience feels.
3. Introduce the product/solution with a clear benefit, not just a feature list.
4. Add social proof or credibility signal if relevant (reviews, numbers, demonstration).
5. End with one clear call-to-action.
6. Break the script into timestamped shot segments (0-3s, 3-8s, 8-15s, etc.) with visual direction for each.
7. Suggest on-screen text overlays for key moments (many viewers watch muted).
8. Keep total runtime 15-30 seconds unless otherwise specified.

## Output format
```
## Hook (0-3s)
Visual: ...
VO/Text: ...

## Problem (3-8s)
Visual: ...
VO/Text: ...

## Solution (8-15s)
Visual: ...
VO/Text: ...

## Social proof (15-20s)
Visual: ...
VO/Text: ...

## CTA (20-25s)
Visual: ...
VO/Text: ...

## On-screen text overlays: [list]
## Suggested platform: [TikTok/Reels/Shorts]
```

## Quality checklist
- [ ] Hook is specific, not generic ("Check this out")
- [ ] Every segment has both visual direction and spoken/text content
- [ ] Single clear CTA at the end
- [ ] Runtime fits platform norms (15-30s)
