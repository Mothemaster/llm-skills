# Order Followup

## Purpose
Use this skill to write clear follow-up messages for order confirmation, shipping updates, delays, and post-delivery check-ins.

## Inputs
- Order status
- Customer issue or trigger
- Tone requirements
- Brand style

## Process
1. State the status clearly.
2. Reduce uncertainty.
3. Add the next action or expected timeline.
4. Keep the tone helpful and calm.

## Output
- Email or message draft
- Optional short and long versions
- Optional apology or recovery wording

## Quality bar
- Be clear, not defensive.
- Respect customer frustration.
- Include the next step.
