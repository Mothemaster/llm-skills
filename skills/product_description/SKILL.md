# Product Description

## Purpose
Use this skill to create product copy for ecommerce, catalogs, landing pages, and marketplaces.

## Inputs
- Product name
- Features/specs
- Audience
- Brand tone
- Optional SEO keyword

## Process
1. Understand what matters most to the buyer.
2. Turn features into benefits.
3. Keep the copy concrete and specific.
4. Make scanning easy.
5. Adapt length to the product page context.

## Output
- Product title
- Short and/or long description
- Bullet highlights
- Optional SEO-friendly version

## Quality bar
- Avoid fluff and generic hype.
- Focus on what helps the buyer decide.
- Match the brand voice.
