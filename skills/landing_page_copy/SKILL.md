# Landing Page Copy

## Purpose
Use this skill to write landing pages that explain the offer clearly and move visitors toward action.

## Inputs
- Offer
- Audience
- Primary CTA
- Proof points
- Optional objections to address

## Process
1. Clarify the offer and audience.
2. Lead with a strong value proposition.
3. Build page flow: hero, benefits, proof, objections, CTA.
4. Keep copy specific and conversion-focused.

## Output
- Headline and subheadline
- Section-by-section copy
- CTA options
- Optional FAQ section

## Quality bar
- Be clear before being clever.
- Reduce friction and doubt.
- Make the CTA obvious.
