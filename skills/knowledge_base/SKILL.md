# Knowledge Base

## Purpose
Use this skill to organize help content, internal docs, FAQs, and reference material into a maintainable knowledge base.

## Inputs
- Existing content
- Audience
- Main use cases
- Search or support goals

## Process
1. Audit current content.
2. Remove duplicates and outdated material.
3. Group by user intent.
4. Standardize structure and naming.
5. Improve findability and update paths.

## Output
- Knowledge base structure
- Category map
- Article templates
- Improvement recommendations

## Quality bar
- Make information easy to find and maintain.
- Write for real questions users ask.
- Reduce overlap and ambiguity.
