# Skill: GEO Optimizer (Generative Engine Optimization)

## Purpose
Optimize content so it gets surfaced, cited, and quoted by AI answer engines (ChatGPT, Perplexity, Google AI Overviews, Claude) — not just ranked by traditional search engines.

## When to use this skill
- User asks to "get found by AI," "show up in ChatGPT answers," "optimize for AI search," or "improve AI visibility."
- Any content task where discoverability via AI assistants matters more than classic SEO rankings.

## Process
1. Identify the core question(s) the target audience would ask an AI assistant related to the topic.
2. Structure content with clear, extractable answers near the top — short, direct, quotable sentences (40-60 words) that AI engines can lift as citations.
3. Use explicit entity names (brand, product, person) instead of vague pronouns, so AI engines correctly attribute facts.
4. Add structured data patterns: definitions, numbered steps, comparison tables, FAQ blocks — AI engines favor structured, scannable formats.
5. Include original data, stats, or unique claims — AI engines prioritize citing sources with distinct, non-duplicated information.
6. Write in a neutral, factual tone for the "answer" sections; save persuasive/brand voice for surrounding context.
7. Add a clear "last updated" date signal in the content structure — freshness affects AI citation likelihood.
8. Avoid keyword stuffing; AI engines reward semantic clarity over repetition.

## Output format
- A GEO-optimized content brief or full page/section including:
  - Target question(s) it answers
  - A citation-ready answer block (40-60 words)
  - Supporting structured sections (FAQ, comparison table, steps)
  - Suggested schema markup type (FAQPage, HowTo, Article)

## Quality checklist
- [ ] Answer to the core question appears within the first 100 words
- [ ] At least one unique data point or claim not found elsewhere
- [ ] Structured format present (list, table, or FAQ)
- [ ] Clear entity names used throughout, no ambiguous pronouns
- [ ] Neutral, factual tone in answer blocks
