# Skill: Image Prompt Designer

## Purpose
Write precise, high-quality prompts for AI image generation tools (Midjourney, DALL-E, Stable Diffusion, Flux) tailored to marketing, product, and brand visuals.

## When to use this skill
- User asks for "an image," "a visual," "a hero image," "product mockup," "ad creative image," or similar.

## Process
1. Clarify the purpose: hero image, product shot, social post visual, ad creative, blog illustration.
2. Define subject clearly and specifically (what/who is in frame).
3. Define style (photorealistic, illustration, 3D render, flat design, editorial photography, etc.).
4. Define composition (camera angle, framing, focal point, aspect ratio).
5. Define lighting and mood (soft natural light, dramatic studio lighting, warm/cool tone).
6. Define color palette if brand-relevant.
7. Add technical modifiers appropriate to the target tool (e.g., "--ar 16:9 --v 6" for Midjourney).
8. Provide 2-3 prompt variations for A/B testing.

## Output format
```
## Prompt 1 (Primary)
[Full prompt text]

## Prompt 2 (Variation - different angle/mood)
[Full prompt text]

## Prompt 3 (Variation - different style)
[Full prompt text]

## Recommended tool: [Midjourney / DALL-E / Flux / SDXL]
## Suggested aspect ratio: [X:X]
```

## Quality checklist
- [ ] Subject, style, composition, lighting all explicitly defined
- [ ] At least 2 prompt variations provided
- [ ] Tool-specific syntax included where relevant
- [ ] No vague adjectives without concrete visual description
