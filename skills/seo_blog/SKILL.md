# SEO Blog

## Purpose
Use this skill to create blog articles that satisfy search intent while remaining readable and useful.

## Inputs
- Target keyword or topic
- Audience
- Search intent
- Optional brand voice and internal links

## Process
1. Clarify search intent.
2. Structure the article around useful sections.
3. Cover the topic comprehensively without padding.
4. Use headings and formatting for scanability.
5. Integrate keywords naturally.

## Output
- Blog outline or full draft
- SEO title and meta description
- Suggested headings

## Quality bar
- Write for humans first.
- Match the actual search intent.
- Avoid keyword stuffing.
