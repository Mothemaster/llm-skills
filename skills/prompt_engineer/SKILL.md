# Prompt Engineer

## Purpose
Use this skill to write strong prompts for content generation, extraction, classification, reasoning, or agent behavior.

## Inputs
- Task goal
- Model behavior needed
- Output format
- Constraints, tone, or refusal rules

## Process
1. Clarify the actual task.
2. Remove ambiguity.
3. Specify the expected output format.
4. Add constraints and examples only when useful.
5. Iterate for consistency and edge cases.

## Output
- Final prompt
- Optional system prompt and user prompt split
- Notes on why the prompt works

## Quality bar
- Prefer clarity over verbosity.
- Make outputs easy to validate.
- Avoid unnecessary prompt ceremony.
