# Skill: Lifecycle Email Writer

## Purpose
Write multi-step email sequences tied to customer lifecycle stages: welcome series, abandoned cart, win-back, post-purchase, re-engagement.

## When to use this skill
- User asks for "welcome email series," "abandoned cart emails," "win-back campaign," "onboarding emails," "retention emails."

## Process
1. Identify the lifecycle stage and sequence length (typically 3-5 emails).
2. Map each email to a specific goal:
   - Welcome: build trust, set expectations, first small action
   - Abandoned cart: remind, address objection, incentivize (in that order across 2-3 emails)
   - Win-back: re-engage lapsed customers with a reason to return (new value, not just discount)
   - Post-purchase: confirm, educate on use, request review/referral
3. Write each email with: subject line, preview text, body copy, single CTA.
4. Space emails logically (e.g., cart abandonment: 1hr, 24hr, 72hr; welcome: day 0, 2, 5, 9).
5. Vary tone slightly per email — first touch warmer/simpler, later touches can be more direct/urgent.
6. Ensure each email in the sequence has ONE goal — don't cram multiple CTAs.

## Output format
```
## Sequence: [Type] — [X] emails

### Email 1 (Send: [timing])
Subject: ...
Preview text: ...
Body: ...
CTA: ...

### Email 2 (Send: [timing])
...
```

## Quality checklist
- [ ] Sequence length appropriate to type (3-5 emails)
- [ ] Send timing specified for each email
- [ ] Each email has exactly one CTA
- [ ] Tone/urgency progresses logically across the sequence
