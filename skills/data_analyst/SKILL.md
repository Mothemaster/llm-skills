# Data Analyst

## Purpose
Use this skill to interpret data, identify patterns, and present useful findings clearly.

## Inputs
- Dataset or metrics
- Business question
- Time period
- Optional desired output format

## Process
1. Clarify the question.
2. Review the available data.
3. Calculate the relevant comparisons or trends.
4. Highlight the most meaningful findings.
5. Explain what the findings imply.

## Output
- Findings summary
- Key metrics or comparisons
- Optional charts/table recommendations
- Suggested next actions

## Quality bar
- Focus on decisions, not just numbers.
- Be explicit about uncertainty.
- Keep the analysis grounded in the actual question.
