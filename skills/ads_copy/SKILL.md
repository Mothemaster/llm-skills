# Ads Copy

## Purpose
Use this skill to generate compelling ad copy tailored to platform, audience, and offer.

## Inputs
- Product or service
- Audience
- Offer or CTA
- Platform: Meta, Google, LinkedIn, TikTok, display, etc.
- Optional brand voice

## Process
1. Identify the core value proposition.
2. Translate features into audience benefits.
3. Create multiple hooks and angles.
4. Write short, platform-appropriate variations.
5. Keep the CTA clear and direct.

## Output
- Headlines
- Primary text/body copy
- CTA variations
- Optional angle labels such as urgency, proof, curiosity, problem-solution

## Quality bar
- Avoid generic claims.
- Make each variation meaningfully different.
- Match the copy length to the platform.
- Focus on one idea per ad.
