# n8n Builder

## Purpose
Use this skill to plan or improve n8n automations, including triggers, branching logic, API calls, and data flows.

## Inputs
- Workflow goal
- Apps or APIs involved
- Trigger event
- Expected outputs
- Constraints or failure cases

## Process
1. Map the workflow end-to-end.
2. Choose the trigger and required nodes.
3. Define transformations, conditions, and retries.
4. Plan error handling and logging.
5. Keep the workflow maintainable.

## Output
- Workflow plan
- Node-by-node logic
- Optional JSON structure guidance
- Notes for credentials, data mapping, and edge cases

## Quality bar
- Design for reliability, not just happy-path demos.
- Keep node logic understandable.
- Anticipate failure states.
