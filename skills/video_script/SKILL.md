# Video Script

## Purpose
Use this skill to write scripts that are easy to speak, visually clear, and structured for retention.

## Inputs
- Video topic
- Platform or format
- Length target
- Audience
- Goal

## Process
1. Start with a strong hook.
2. Keep one clear message.
3. Build logical progression.
4. Write for spoken delivery.
5. End with a CTA or memorable close.

## Output
- Full script
- Hook options
- Optional shot or scene notes

## Quality bar
- Make it sound like speech, not an essay.
- Keep pacing tight.
- Optimize for viewer retention.
